1. 全网抓取简介

全网抓取,旨在简化抓取过程,不需要要vertical抓取那样复杂;同时,全网对于特定的信息的抓取也没有vertical那么快;对于哪些页面该访问,哪些页面不该访问,没有好的控制,会出现有的信息没有成功被抓取的情况; 像这两个需求,就比较适合全网抓取: 1.不停地访问某网站,比如说吧,techweb.com,全站访问,一旦发现有"联想","legend"这个词,就将地址记入数据库; 2.访问xinhuanet.com,将所有正文页(地址类以xinhuanet.com/{year}/{month}/content-****.html)的正文内容记录下来,提取正文,存入数据库;
