About Hyer

Hyer is a vertical search cralwer library written in python. It provides a number of methods to mine data from kinds of sites.

Features

  1. robots.txt protocol supported;
  2. by setting how many visits per minute ,you can avoid to visit site too frequencly;
  3. you can setup proxies to tell target web server various IP addresses;
  4. cache URL 's HTML;
  5. extract MainText from HTML by specific a * link-threshold *
  6. visit sites with cookie;

关于worker类

worker是完成指定的一系列操作,达到某个效果的一个过程。每个worker都有一个叫做"职位"的属性,我用post来表示了.在实际运行中每一类worker可能会有很多个在同时运行;

特性

  1. 支持robots.txt
  2. 可以在抓取过程中设置每分钟访问该站点多少次
  3. 可以设置代理
  4. 可以根据链接的密度获得正文
  5. 带cookie访问站点
